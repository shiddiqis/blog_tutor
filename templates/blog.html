Berbagai Cara Screenshot di Laptop Dell, Sudah Coba?

Buat sebagian orang, aktivitas screenshot sudah jadi kegiatan sehari saat bekerja. Mulai dari mengabadikan pertemuan daring hingga mengambil gambar untuk mencari inspirasi desain. Salah satu perusahaan laptop ternama asal Amerika, Dell, menyediakan beberapa cara screenshot di laptopnya untuk kepuasan para penggunanya.

Untuk kamu yang belum mengetahui cara screenshot di laptop Dell, jangan khawatir karena beberapa metode akan dijelaskan dibawah ini.

Screenshot dengan kombinasi tombol

Kamu juga dapat melakukan screenshot dari jendela atau halaman yang sedang aktif, berikut cara melakukannya.

  • Tekan tombol Alt dan Print Screen atau tombol PrtScn pada keyboard secara bersamaan. Hal ini untuk menangkap jendela atau halaman dan menyimpannya ke clipboard komputer
  • Klik tombol Start di pojok kiri bawah layar dan ketik "paint"
  • Buka program Paint dan tempel tangkapan layar ke dalam program dengan cara menekan tombol Ctrl dan V pada keyboard secara bersamaan
  • Jika gambar dianggap terlalu lebar, gunakan fitur Pangkas atau Crop untuk memotong gambar. Pada proses ini kamu juga dapat menambahkan teks, simbol, dan warna.
  • Simpan gambar dengan cara menekan CTRL+V atau klik ikon disket di pojok kiri atas.

Screenshot dan simpan otomatis

Jika kamu menggunakan Windows 8 atau Windows 10, ada beberapa cara yang lebih sederhana tanpa menggunakan Snagit.

  • Tekan logo Windows atau tombol Win dan tombol PrtScn pada keyboard secara bersamaan. Pada proses ini layar kamu akan meredup beberapa detik.
  • Buka fitur Explorer dan tekan tombol logo Windows dan E pada keyboard secara bersamaan.
  • Buka file C: \Users\ username\Pictures\Screenshots dan kamu akan temukan hasil screenshot-nya.

Screenshot menggunakan Sniping tool

Fitur screenshot ini adalah fitur yang disediakan oleh windows yang dapat ditemukan di semua versi OS Windows. Kamu dapat menggunakan fitur ini dengan cara:

  • Tekan tombol Windows pada keyboard
  • Ketik Snipping Tool pada kotak pencarian
  • Pilih fitur screenshot yang diinginkan seperti, rectangular, window, atau full screen snip dengan cara menekan tombol panah di sebelah tulisan New
  • layar akan membeku dan kamu dapat memilih area mana saja yang ingin di-screenshot.
  • Edit gambar sesuai keinginan kamu. jika sudah selesai tekan Ctrl + S atau tekan ikon disket di pojok kiri atas.

Screenshot menggunakan Snagit

Snagit adalah program yang menyediakan fitur screenshot dengan pengeditan gambar dan perekaman layar. Selain itu kamu juga dapat mengedit hasil screenshot dengan mudah menggunakan fitur pengeditan gambar yang canggih, bahkan kamu juga dapat merekam video. Berikut cara screenshot menggunakan Snagit.

  • Masuk ke halaman yang ingin ada screenshot
  • Buka program Snagit, lalu tekan tombol Capture.
  • Arahkan kursor ke wilayah atau kolom gambar untuk memilih area secara otomatis, atau kamu dapat memilih area kustom dengan cara klik dan menyeret kursor sesuai gambar yang diinginkan.
  • Setelah kamu menentukan gambar atau area mana yang akan di screenshot, klik ikon kamera untuk menyimpan hasil screenshot sebagai gambar.
  • Edit gambar di program Snagit sesuai dengan selera. Dalam proses pengeditan ini kamu dapat menambahkan bentuk, teks, efek, atau membuat penyesuaian pada gambar.
  • Simpan hasil screenshot dengan menekan Ctrl + S atau klik tombol Share di pojok kanan atas untuk membagikannya.

Itulah beberapa cara mudah untuk screenshot di laptop Dell, selamat mencoba!
